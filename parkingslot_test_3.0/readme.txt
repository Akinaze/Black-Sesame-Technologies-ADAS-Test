输出：
1. test_result.csv
说明：包含三种车位类型的真值总数、检测数、类别匹配数、类别不匹配数、漏检数、检出率、类别匹配率
###三种车位类型：垂直车位：reversing，水平车位：parallel，斜车位：oblique，人工漏标注：no anno
###真值总数：total num，检测数：detect num，类别匹配数：type match num，类别不匹配数：type mismatch num，漏检数：miss num
###检出率：detect rate，类别匹配率：type match rate

2. miss_path_record.txt
说明：包含所有含有漏检车位的原图的路径。

3. boxes文件夹
说明：其中包含每个匹配上的timestamp的dt和gt的车位图片。
